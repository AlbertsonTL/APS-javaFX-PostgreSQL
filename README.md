🛠️ Proyecto de Inspección y Corrección de un Sistema Punto de Venta (JavaFX)

📋 Descripción

Este proyecto tiene como objetivo realizar una inspección técnica de una aplicación Punto de Venta (POS) desarrollada en JavaFX, identificando y corrigiendo fallas introducidas intencionalmente para restaurar su funcionamiento.

Durante el desarrollo del proyecto se analizaron errores tanto sintácticos como lógicos, aplicando las correcciones necesarias para garantizar la estabilidad y el correcto funcionamiento de la aplicación.

🏗️ Tecnologías utilizadas

- Java FX – Desarrollo de la interfaz gráfica.
- PostgreSQL – Sistema de gestión de base de datos.
- JDBC – Conexión entre la aplicación y la base de datos.

📂 Estructura del proyecto

El proyecto está compuesto por:

- Aplicación Punto de Venta desarrollada con JavaFX.
- Base de datos PostgreSQL.
- Script de respaldo (backup) para restaurar la base de datos en un entorno de pruebas.

🎯 Objetivos

- Inspeccionar el código fuente de la aplicación.
- Identificar errores sintácticos y lógicos.
- Corregir las fallas detectadas.
- Restaurar el funcionamiento completo del sistema.
- Documentar las correcciones realizadas.

🚀 Puesta en marcha

1. Restaurar la base de datos utilizando el script de respaldo proporcionado.
2. Configurar los parámetros de conexión a PostgreSQL.
3. Importar el proyecto en un IDE compatible (IntelliJ IDEA, NetBeans o Eclipse).
4. Compilar y ejecutar la aplicación.
5. Verificar que todas las funcionalidades del sistema operen correctamente después de las correcciones.

📌 Nota

Este proyecto fue desarrollado con fines académicos como práctica de inspección de software, depuración de código y mantenimiento de aplicaciones Java.
